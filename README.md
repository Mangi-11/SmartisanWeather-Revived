# Smartisan Weather Revived

锤子天气 8.1.3 的现代 Android 复刻项目。在保留原版视觉与交互体验的基础上，使用当前 Android 技术栈重新实现。

## 技术栈

- Kotlin + XML Layout / Android View
- Room、DataStore、Coroutines、StateFlow
- 小米天气中国区混合数据源
- 最低 Android 8.0（API 26）

## 构建

准备 JDK 17 和 Android SDK，然后执行：

```bash
./gradlew assembleDebug
```

Debug APK 位于 `app/build/outputs/apk/debug/`。
