一、 我们如何收集和使用个人信息

为了向你提供我们的服务，以及为你带来更好的用户体验，我们可能会从你的设备收集一些信息和基本使用数据。你有权自行选择是否提供我们请求的信息，但在一些情况下，如果你拒绝，我们可能无法向你提供相应的服务。

我们收集的信息取决于你与 Smartisan 互动的环境、你所做的选择，包括你的隐私设置以及你所使用的产品和功能。

（一）我们收集个人信息

根据你选择的服务，我们可能收集以下信息中的一种或多种：

  1. 你主动提供给我们的信息
    1. Smartisan 账号相关信息：姓名、生日、手机号码、电子邮箱、Smartisan 账号用户名和密码、照片。
    2. 订购产品、申请退货或退款或使用收费服务涉及的订单信息：产品/服务名称、收件人姓名、收件人联系方式、交货地址、支付方式及账号、与客服联系及交流的记录。
    3. 通信、共享内容：我们的某些服务允许你与其他人通信、共享。出于技术需要，通信、共享内容可能会经过加密后通过我们的系统进行传输。如你通过云服务同步你上传的某些信息或者通过分享功能上传或下载你选定的信息时，数据会在加密后通过我们的系统进行传输。
    4. 面部特征信息或你的指纹信息。 你的面部特征信息或你的指纹信息将会被存储在设备本地安全模块或隐私模块，系统及应用均无法获取该信息。
    5. 语音信息。你使用语音命令控制服务或其他语音服务时，需要开启麦克风权限。如当你使用无障碍、大爆炸等功能或系统应用的语音文字转化服务时，第三方语音服务商可能需要收集、使用相关信息以便实现为你提供相关语音服务的目的。我们不会存储你的录音。当你使用语音备忘录（如闪念胶囊）等功能自行上传录音，并选择使用云服务同步功能的，你的录音将会被保存在你的云存储空间，我们不会查看你的录音，也不会以任何形式使用你的录音内容。你可以在语音备忘录中自行管理或删除你的录音。
    6. 其它我们可能会要求你提供和收集你的个人信息的情况，包括 Smartisan 设备中存储的并经你同意由我们收集或由你上传的联系人信息、日程信息、参与抽奖或比赛、参加我们自办或业务合作伙伴组织的推广活动或营销活动、填写问卷调查表、参加我们自办或业务合作伙伴主办的用户论坛或博客，具体信息内容请以当时向你告知的为准。
  2. 我们在你使用服务过程中收集的信息

    3. 除了你提供的信息以外，我们还可能通过 Smartisan 设备中的软件以及其他方式来收集你使用我们的服务的相关信息。例如，我们可能收集：

    1. 设备信息及 SIM 卡相关信息：设备名称、设备型号、地区和语言设置、设备识别码（IMEI、Android ID 和 SN）、设备硬件信息与状态、操作系统版本、SIM 卡运营商和归属地。
    2. 日志信息：与你使用某些功能、应用、网站相关的信息，包括使用服务的时间和持续时间、IP 地址、网络请求信息、临时消息历史、标准系统日志、错误崩溃信息、使用服务或对系统进行操作而产生的行为日志信息以及设备和软件的事件信息（如注册时间、访问时间、活动时间等）。
    3. 位置信息：设备的 GPS 信号或有关附近 WiFi 接入点和手机信号塔的信息、设备位置信息、网络服务提供商信息 。我们将询问你希望启用的基于位置的应用程序。你可以从设备设置中修改你设备的位置设置，例如更改或禁用基于位置服务的服务所使用的方法和服务器、或修改位置信息的准确性，从而更改提供给 Smartisan 的位置信息。
    4. 你上传到 Smartisan 云的信息。你上传到 Smartisan 云服务的信息会安全存放在云端，以实现你在不同应用、不同设备之间的共享。我们不会查看你上传的信息。
    5. 与你的应用使用相关的信息：包括应用基础信息，例如应用列表、应用 ID 信息、SDK 版本、系统更新设置、应用设置（地区、语言、时区、字体），以及应用状态记录（例如下载、安装、更新、删除）。
  3. 来源于第三方的信息

    4. 在法律允许的情况下，我们可能从公共或商用来源获取有关你的数据，并可能将其与收到的或与你有关的其他信息结合。

    1. 出于安全和防欺诈的目的，针对特定账号、网络交易等服务，在你授权的前提下，通过合法来源核实你的信息（如电话号码）；
    2. 我们还可能从第三方社交网络服务中获取某些信息，例如账户、昵称、头像（例如，当你使用社交网络账户登录我们的服务时）；
    3. 他人提供的与你有关的信息，例如其他用户在使用锤子商城时填写的你的收货地址。
  4. 非识别性信息的收集和使用

    5. 我们还可能收集其他无法识别到特定个人、根据当地适用法律不属于个人信息的信息，例如你使用特定服务时产生的统计类数据，如匿名设备相关信息，其中包含应用启动数、页面访问数、重点按钮点击数、设置项的配置属性、日活事件、页面访问事件、页面访问时长事件；网络监控数据如请求时长、请求与错误请求数等；以及应用崩溃事件。收集此类信息的目的在于改进产品的运行效率，改善我们向你提供的服务。所收集信息的类别和数量取决于你如何使用我们产品或服务。

    我们会将信息汇总，用于帮助我们向客户提供更有用的信息，了解客户对我们网站、产品或服务中的哪些部分最感兴趣。如果我们将非个人信息与个人信息结合使用，则在结合使用期间，此类信息将被视为个人信息。

  1. 激活、注册你所购买的 Smartisan 产品或服务 为正常使用 Smartisan 产品或服务，你需要激活产品或注册账号，我们将自动收集设备和 SIM 卡信息。电话号码、位置区域代码等位置信息用于用户认证和服务激活。
  2. 创建和维护你 Smartisan 账号 为创建“Smartisan ID”，你需提供手机号码、电子邮箱用于创建你的 Smartisan 账号和密码。你还可根据情况添加你的生日、地址等个人信息用以完善你的个人资料。如你不提供【手机号码、用户名及密码】则可能无法注册 Smartisan 账号，享受相应服务。
  3. 获得保修服务 为保证你可以及时方便的获得保修服务，你在创建你的 Smartisan 账号后我们将自动获取【设备信息】以获取保修卡。
  4. 快速解锁设备 面部识别和指纹解锁可以使用预先记录的面部或指纹信息以帮助你快速解锁设备。要实现前述目的，你需要从不同角度提供你的面部特征信息或你的指纹信息，上述信息将留存在你的 Smartisan 设备本地安全模块或隐私模块，系统及应用均无法获取。
  5. 语音控制辅助功能 你使用语音命令控制服务或其他语音服务时，需要开启麦克风权限。如你使用无障碍、大爆炸等功能或系统应用的语音文字转化服务时，第三方语音服务商可能需要收集、使用相关信息以便实现为你提供相关语音服务的目的。我们不会存储你的录音。如你使用语音备忘录（如闪念胶囊）等功能自行上传录音，并选择使用云服务同步功能的，你的录音将会被保存在你的云存储空间，我们不会查看你的录音，也不会以任何形式使用你的录音内容。你可以在语音备忘录中自行管理或删除你的录音。
  6. 骚扰电话及短信拦截服务 当你主动开启并使用骚扰电话及短信拦截服务时，我们或第三方服务商将需要读取你提供需要拦截的手机号码及短信信息，且我们和/或第三方服务商将对你接收的电话或短信所对应的号码进行收集查验，如你拒绝可能无法使用该服务。
  7. 输入法服务 为了保障服务与系统兼容稳定，我们或第三方输入法技术服务商将会自动收集你的设备信息。
  8. 处理你的订购单 我们将使用订单相关的信息包括收货人姓名、收货地址、收货人联系电话号码，处理订购单和相关的售后服务，包括客户支持、订单配送、发票打印等。订单号将用于校验物流合作伙伴的订单和包裹的实际交付。收件人信息包括姓名、地址、电话号码和邮政编码将用于送货。购买物品的清单用户打印发票并使你核对确认包裹中的物品。

  18. 参与锤子科技论坛 与锤子科技论坛或其它 Smartisan 相关的社交平台上的个人信息，如你的昵称和头像，可被用于与其他用户进行互动，参与论坛。

（四）依法豁免征得同意收集和使用的个人信息

请你理解，在下列情形中，根据法律法规及相关国家标准，我们收集和使用你的个人信息无需征得你的授权同意：

  1. 与国家安全、国防安全直接相关的；
  2. 与公共安全、公共卫生、重大公共利益直接相关的；
  3. 与犯罪侦查、起诉、审判和判决执行等直接相关的；
  4. 出于维护个人信息主体或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；
  5. 所收集的你的个人信息是你自行向社会公众公开的；
  6. 从合法公开披露的信息中收集的你的个人信息的，如合法的新闻报道、政府信息公开等渠道；
  7. 根据你的要求签订或履行合同所必需的；
  8. 用于维护软件及相关服务的安全稳定运行所必需的，例如发现、处置软件及相关服务的故障；
  9. 为合法的新闻报道所必需的；
  10. 学术研究机构基于公共利益开展统计或学术研究所必要，且对外提供学术研究或描述的结果时，对结果中所包含的个人信息进行去标识化处理的；
  11. 法律法规规定的其他情形。

二、 如何使用 Cookie 和同类技术

Smartisan 的网站、移动应用软件、在线服务、电子邮件及广告可能会使用 Cookie、像素标签和网站信标等本地存储技术，用于分析趋势、管理网站、追踪网站活动并收集关于用户群体的统计信息，帮助我们更好地提供服务，例如使我们了解你浏览我们网站的那些部分，以衡量网络搜索的效果并加以改善。

（一）Cookie

Cookie 是从网络服务器发送到你的移动设备浏览器、存储在你移动设备上的纯文本文件。

Smartisan 有时会在计算机或移动设备上存储 Cookie，我们启用 Cookie 的目的包括：

  1. 账号登录和验证。当你使用 Smartisan 账号登录网站时，借助于 Cookie，你可以直接从网站的一个页面跳转到另一个页面，而不必在每个页面上重新登录；
  2. 存储你的偏好和设置，帮助你更轻松访问或使用服务。网站能够借助 Cookie 来保存设置，如计算机或移动设备的语言、字体大小、购物车和其他浏览偏好等，帮助你省去重复填写的步骤；
  3. 向你推荐、展示基于兴趣的内容或广告。Smartisan 使用 Cookie 来收集有关你在线活动的信息并发现你的兴趣所在，并向你提供与你相关度更高的内容或广告。
  4. 统计分析。借助于 Cookie，Smartisan 可以收集你使用我们网站和其他应用程序的信息，如记录用户的单次访问（使用会话 Cookie）或多次访问（使用永久 Cookie）。 Smartisan 不会将 Cookie 用于本声明所述目的之外的任何用途。你可根据自己的偏好管理或删除 Cookie。有关详情，请参见 AboutCookies.org。

你可以清除计算机上保存的所有 Cookie，大部分网络浏览器都设有阻止 Cookie 的功能。如需详细了解如何更改浏览器设置，请访问以下链接：Internet Explorer、Google Chrome、Mozilla Firefox、Safari 和 Opera。

如果你清除 Cookie，则需要在每一次访问 Smartisan 的网站时亲自更改用户设置。同时也请注意，Smartisan 的某些服务可能必须使用 Cookie，禁用 Cookie 可能会影响你使用这些服务的全部或部分功能。

（二）网站信标/像素标签

网站信标通常是一种嵌入到网站或电子邮件中的电子图像，可用于在你查看网页或电子邮件时识别你设备上的 Cookie。像素标签使我们可以用客户可阅读的格式发送电子邮件，并告知我们电子邮件是否被打开。我们可以利用这些信息来减少或者免除向客户发送电子邮件。

个人信息：指以电子或者其他方式记录的能够单独或者与其他信息结合识别特定自然人身份或者反映特定自然人活动情况的各种信息。

个人敏感信息：指包括身份证件号码、个人生物识别信息、银行账号、财产信息、行踪轨迹、交易信息、儿童信息等的个人信息。

去标识化技术:指通过对个人信息的技术处理，使其在不借助额外信息的情况下，无法识别个人信息主体的过程。

匿名化：指通过对个人信息的技术处理，使得个人信息主体无法被识别，且处理后的信息不能被复原的过程。
